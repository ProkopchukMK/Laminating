Необхідно для роботи

1. Windows 7 х64 або вище.
2. Втановлений .net 4.5.2 що знаходиться в папці DotNetFX452 ( файл NDP452-KB2901907-x86-x64-AllOS-ENU) або включені авто оновлення у Windows. 
3. Зєднання до SQL Server 2008 або вище.
4. Потрібно надати доступ Laminatsia всім користувачам хто буде користуватися базою даних.
5. Потрібні дозволи для користувачів бази даних public,db_owner,db_datawriter, db_datareader.
6. Запускати програмку від адміністратора.

Налаштування програми.
1. Після встановлення програми, повинні запустити програму ввівши логін Адміністратор та пароль 1, вибрати роль Адміністратори. 
2. Натиснути вхід та чекати повідомлення про налаштування зєднання.
	2.1 Якщо у вас вже є база даних то просто за допомогою списку чи введеням вручну сервера підключаємся до серверу та до бази даних та зберігаємося. 	
	2.2 Якщо у вас не має бази даних, то користуємося вкладкою Створити базу даних.
	2.3 Після створення, перезапускаємо програму, та аналогічно вводимо логін, пароль та роль - програма повинна запуститися. 
		2.2.1 Потім додаємо користувачів та змінюємо пароль для Адміністратора.
3. Також налаштування програмки можна за допомогою файлів програми. Для доступу до бази даних Laminatsia.exe.config, який лежить у папці з Laminatsia.exe фалом. C:\Program Files (x86)\Prokopchuk\Laminatsia (для х64 версій)
4. Створити базу можна і скріптом з файлу CreateDataBaseLaminatsia який знаходиться в папці з встановленою програмкою.

Питання та вирішення.
1. Не встановлюється програма.

1.1 Або програма вже встановлена, або її потрібно видалити, бо вона встановилася не правильно.
1.2 Не становлений .net 4.5.2. Або беремо з папки програми або скачуємо з сайту https://www.microsoft.com/en-US/download/details.aspx?id=42642.

2. Не підключається до серверу.

2.1 Не налаштоване локальне зєднання. (Перевірити доступність серверу через Провідник)
2.2 Не налаштований SQL Server. (Перевірити відкритість портів та доступність користувачів)
2.3 Не наданий дозвіл користувачу. (Знайти користувача на SQL Server та надати дозвіл до бази даних)

Якщо всеж таки у вас не виходить, значить у вас криві руки або вони ростуть не як у системного адміністратора і потрібно звати системного адміністратора з нормальними руками.
